## Adding Vite and React Apps to GitHub

1. **Initialize a Git Repository**  
   Run the following command in your project directory to initialize a Git repository:

   ```bash
   git init
   ```

2. **Create a `.gitignore` File**  
   Add a `.gitignore` file to exclude unnecessary files. For example:

   ```
   node_modules/
   dist/
   .env
   ```

3. **Commit Your Code**  
   Stage and commit your files:

   ```bash
   git add .
   git commit -m "Initial commit with Vite and React apps"
   ```

4. **Create a GitHub Repository**  
   Go to [GitHub](https://github.com) and create a new repository. Copy the repository URL.


6. **Verify on GitHub**  
   Visit your GitHub repository to confirm the code has been uploaded.
